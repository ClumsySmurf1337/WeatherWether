# Art Direction

## Visual Pillars

- Calm but dramatic weather mood.
- Readable puzzle state above all aesthetics.
- Mobile-first clarity with desktop polish parity.

## Style Language

- Clean, flat-shaded 2D tiles with subtle gradients.
- Soft rounded corners for interactable UI.
- High-contrast weather overlays for readability.
- Limited palette per biome to avoid visual noise.

## Base Palette Targets

- Rain: slate blues and desaturated teals.
- Sun: warm ambers and pale yellows.
- Frost: cool cyans and bright icy highlights.
- Fog: neutral grays with low-opacity layering.
- Lightning: sharp white-violet accents.
- Wind: light directional strokes, low opacity.

## Asset Specs

- Tile size baseline: `64x64` source, scalable.
- Weather card icons: `256x256`.
- UI icons: `128x128`.
- Export PNG with transparent background.

## Gameplay Readability Rules

- Terrain states must be visually distinct at 2x zoom out.
- Never rely on color alone for state (add shape/pattern cue).
- Animations must not hide path-critical tiles.

# Stitch Prompt Pack (Mobile UX)

Tool reference: [Stitch](https://stitch.withgoogle.com/)

Use these prompts to generate UX wireframes and references for implementation.

## Prompt 1: Onboarding + Main Menu

```text
Design a mobile-first UX flow for a premium puzzle game called Whether.
Create wireframes for:
1) First launch onboarding
2) Main menu
3) Continue game and settings access
Style: clean, modern, atmospheric weather theme
Constraints: one-hand use, large tap targets, minimal cognitive load
```

## Prompt 2: World Select + Level Grid

```text
Generate mobile UI wireframes for world select and level selection.
Need 6 worlds, each with a progress indicator and lock state.
Level grid should clearly show completion status and hints.
Include portrait layout first and desktop adaptation notes.
```

## Prompt 3: In-Game HUD

```text
Create in-game HUD wireframes for a grid puzzle game with weather cards.
Elements:
- Card hand tray
- Undo/reset controls
- Objective panel
- Pause/settings access
Design for touch comfort, avoid accidental taps near edges.
```

## Prompt 4: Accessibility Pass

```text
Produce an accessibility-focused UI variant for the existing puzzle game wireframes.
Include larger text option, high-contrast mode, and icon-shape redundancy for colorblind users.
Keep hierarchy and interactions unchanged.
```

# Release Checklist

## Windows/Steam (Current Priority)

- [ ] Core campaign levels validate.
- [ ] Save/load regression pass complete.
- [ ] Mobile-first UI still readable in desktop builds.
- [ ] Build and smoke test on clean Windows machine.
- [ ] Store metadata and screenshots prepared.

## Android (Compatibility Lane)

- [ ] Touch controls verified on target aspect ratios.
- [ ] Performance profile within acceptable limits.
- [ ] Packaging/signing process documented.

## iOS (Deferred Lane)

- [ ] Remote macOS build path approved.
- [ ] Signing materials and secret handling in place.
- [ ] CI template activated only when ready.

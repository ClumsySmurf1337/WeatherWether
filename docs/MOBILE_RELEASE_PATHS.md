# Mobile Release Paths

## Current Path

- Primary now: Windows/Steam builds and gameplay loop.
- Architecture remains mobile-first: touch UI and performance constraints are always considered.

## Android on Windows

- Supported directly from Windows workflows.
- Use mobile preview checks and device validation before release.

## iOS Constraint

- iPhone distribution requires macOS + Xcode toolchain for signing/submission.
- This project includes future-ready templates but does not force iOS execution now.

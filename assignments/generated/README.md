# Generated PM assignments

Markdown here is **created by** `npm run linear:pm-assignments` from your Linear workspace (Backlog, Todo, In Progress). Files are **gitignored** so they stay local and fresh.

Regenerate after board changes:

```powershell
npm run linear:pm-assignments
```

To set **priorities + empty assignees** from the phase plan:

```powershell
npm run linear:pm-organize
npm run linear:pm-organize -- --apply
```

See `docs/PM_AGENT_LINEAR.md`.
